## [1.0.0] - 2026-02-01

### Added
- Initial release of ISTS Protocol
- Core text compression engine with global ledger deduplication
- Image optimization engine using Canvas WebP encoding
- IndexedDB storage utilities
- Zero-dependency implementation with embedded LZ-String
- Comprehensive API for text and image compression
- TypeScript type definitions
- Technical whitepaper
- Complete documentation and examples

### Features
- 80-95% compression ratio on various data types
- Lossless text deduplication
- Perceptual image vectorization
- Cross-session ledger persistence
- Browser-compatible (Chrome 80+, Firefox 75+, Safari 13+, Edge 80+)
